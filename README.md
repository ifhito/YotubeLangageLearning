# Youtubeの動画を用いた、語学学習アプリケーションの作成

## 概容

Youtubeの動画のCaptionを動画の下に表示し、動画を聞き取りながらCaption内の単語を穴埋する語学学習うのためのWebアプリケーション

Reactを勉強するため、内部はReactとReactRoutorを用い記述している。
